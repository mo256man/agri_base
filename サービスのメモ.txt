agri.dbのconfigテーブルにrestart=1を追加する

cronを設定
sudo crontab -e

内容
0 3 * * * /usr/bin/python3 /home/tab/agri/autorestart.py

10分ごとに実行
*/10 * * * * /usr/bin/python3 /home/tab/agri/autorestart.py


サービス開始
sudo systemctl start cron


ブラウザ起動
sudo nano /etc/xdg/lxsession/LXDE-pi/autostart

内容
@lxpanel --profile LXDE-pi
@pcmanfm --desktop --profile LXDE-pi
@xscreensaver -no-splash
@xset s off
@xset s noblank
@xset -dpms
@chromium-browser --hide-crash-restore-bubble http://0.0.0.0:5000/



サービスを作る
sudo nano /etc/systemd/system/agribecky.service


指定の場所に動かす
sudo mv agribecky.service /etc/systemd/system

サービスを起動する
sudo systemctl daemon-reload
sudo systemctl start agribecky
sudo systemctl stop agribecky

確認
sudo LANG=C systemctl status -l agribecky

常駐化
sudo systemctl enable agribecky

DB browser for sqlite
sudo apt-get install sqlitebrowser

IPAフォントをインストール
sudo apt install fonts-ipaexfont
